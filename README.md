# 🍽️ Recipe App (Flask)

A simple web application built with **Flask** that allows users to create, view, search, and manage recipes.  
The app also supports user authentication for features like meal planning, updating, adding, and deleting recipes.

---

## 🚀 Features

- User authentication (login / logout)
- Add, view, and search recipes
- View recipes by category
- Retrieve recipes by ID
- Weekly meal planning
- Responsive UI using Bootstrap
- Server-side rendering with Jinja templates

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, Bootstrap 4.3.1, JavaScript
- **Templating:** Jinja2
- **Database:** SQLite (via SQLAlchemy)
- **Auth:** Flask-Login, JSON web token
- **Documentation:** SwaggerUI
- **Hosting Application**: uWSGI, NGINX, Raspberry Pi
